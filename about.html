关于

这里是一些有关本站的信息

毕竟首页都放个人简介了，这里就放站点说明文吧

域名含义

即 小熊座 的希腊古名拼写，呼应在用 ID: froseiun (from 星雲)，及常用昵称“熊”

而小熊座 α 星恰好为当前的北极星；在极北的夜空闪烁，指引正确的方向

搭建环境

如你所见，全站均为纯手工打造，是我学习前端的练手项目

无流量统计脚本，也无任何冗余插件加载 (其实是我能力有限)

目前是基于 GitHub 仓库 + Cloudflare Pages 自动部署，全站除留言版外均为纯静态页面

对，连 JavaScript 都没有，因为我还不会写🙃，真正意义上的极简个人页

字体

标题使用 Overpass 的 regular 字号

是美国联邦高速公路字体 Highway Gothic 的开源实现版本，非常简洁易读

而正文部分直接使用系统默认字体，因为包含中文字形后，无论是外部引用还是自带字体，都面临过于臃肿的问题

作为对比，Overpass 仅占用 69KB 的空间，不如索性把字体选择权交给用户

图标

感谢设计师的辛勤付出，首页部分的社交图标使用 Remix Icon 所设计的图标

我认为比较符合全站的整体风格，同时有比较宽松的使用许可 (CC-BY 4.0)

文章页

分页下使用 Python 编写的 Pelican 生成文章

主题同样是纯手工打造的，与主站风格保持一致，如遇样式错误请在下方留言让我知道
